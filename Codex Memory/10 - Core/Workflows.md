# 常用工作流

## 任务开始

1. 读取 [[../00 - Start Here|记忆入口]]。
2. 读取 `Rules`、`Preferences`、`Workflows`、`Decisions` 中与当前任务相关的内容。
3. 根据当前工作目录匹配 `20 - Projects` 中的项目页；没有对应页时不自动创建，除非本次产生了值得长期保留的项目背景。
4. 检查项目自己的 `AGENTS.md` 或同类规则；更具体、更新的项目规则优先。
5. 开始任务，并在出现冲突或不确定记忆时向用户核实。

## 任务结束

1. 判断本次是否产生长期有效的规则、偏好、可复用流程、决策或项目背景。
2. 有则更新原条目并注明日期、来源和状态；没有则不写入。
3. 尚未确认但可能有价值的内容放入 `90 - Inbox`，不要直接当成事实。
4. 不保存秘密或大段聊天原文。
